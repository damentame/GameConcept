# Guardians of Balance

A browser-based strategy progression game intentionally designed with ethical systems:

- Transparent, fixed mission rewards
- No loot boxes or hidden odds
- No offline decay or punishment for taking breaks
- Finite 10-mission campaign with a clear ending
- Full tactical board with playable characters you can move and place
- Rich micro-animations on movement, logs, actions, stats, and UI transitions

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Gameplay basics

1. Start the campaign.
2. Select a hero from the roster and click tiles on the tactical field to move.
3. Place at least one hero on a green objective zone.
4. Deploy missions for deterministic rewards and objective bonuses.
5. Use Rest and workshop upgrades to progress through all 10 missions.
